Journey is a Smart Travel Planner Application which satisfy following objectives

1. Design and implement a system that can dynamically generate an itinerary.
2. Automate the process of tour planning based on the preferences of the user.
3. Design and implement a system that can generate an itinerary based on
    multiple users (a group).
4. Provide as a reference tool to view all information about the trip.
5. Provide a storage tool to store the trip history of a specific user.
